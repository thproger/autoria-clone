# Autoria clone
Ендпоінти API описані у файлі (autoria-clone.postman_collection.json)

Для зручності можна у postman можна створити користувача та додати його токен до змінної token, він автоматично заповниться до поля авторизації всіх ендпоінтів.

Для пошти використовується SMTP сервер mailtrap(на жаль, він любить не працювати)

База перестворюється щоразу при запуску для зручності тестування, але є файл міграцій для liquibase.
